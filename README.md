# Velocity Ventures Vietnam — Stealth Landing

Static stealth landing page for [velocityvv.com](https://velocityvv.com).

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

This is a static site. Deploy the repository root to any static host (Vercel, Cloudflare Pages, Netlify, etc.). No build step required.

## Structure

```
public/
  index.html        # Landing page (from VVV Stealth Landing.html)
  assets/           # Static assets (Kamereo icon, etc.)
reference/          # Brand guidelines, design system, and source screenshots
```
